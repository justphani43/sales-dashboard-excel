# 📊 Sales Data Analysis Dashboard (Excel)

## 📌 Project Overview
This project focuses on analyzing sales data to uncover meaningful business insights. An interactive dashboard was developed using Excel to track sales performance, identify trends, and support data-driven decision-making.

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Pivot Tables  
- Data Visualization (Charts & Slicers)

## 🔄 Data Preparation
- Cleaned and transformed raw data using Power Query  
- Handled missing and inconsistent values  
- Structured data for efficient analysis  

## 📊 Key Insights
- Identified overall sales trends across different time periods  
- Analyzed top-performing products contributing to revenue  
- Evaluated regional performance to highlight high and low sales areas  

## 📈 Dashboard Features
- Interactive dashboard with dynamic filtering using slicers  
- User-friendly design for easy exploration of data  
- Visual representation of KPIs such as sales, profit, and product performance
